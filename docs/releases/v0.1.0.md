# SILK v0.1.0

SILK v0.1.0 turns the initial public snapshot into an inspectable OSS release: parsed V1 validation, a generated evidence world, unified CI, contribution paths, and a reproducible V2 candidate.

## Install and try it in five minutes

Use Node.js 20 or newer.

```powershell
git clone https://github.com/ugin-man/SILK.git
cd SILK
npm ci
npm run validate:demo
```

Optional V2 checks:

```powershell
npm run test:v2
npm run audit:v2
```

Open [`v2.next/SILK-V2.html`](../../v2.next/SILK-V2.html) locally to inspect the single-file V2 application.

## Known limitations

Glass Tide shows that a small revision 2 world can be generated, recorded, and structurally validated. It does not prove long uninterrupted generation, book-scale maintenance, or human-approved creative excellence. Multiple independent human evaluations and broader model/environment reproduction remain open in the long-run validation Issue above.

Structural validation does not prove creative quality.
